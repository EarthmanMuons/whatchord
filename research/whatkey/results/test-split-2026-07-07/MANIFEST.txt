WhatKey one-shot test result manifest
=====================================

These committed artifacts are the source of the headline table in
research/whatkey/README.md.

MANIFEST.json is the machine-readable source for the headline report paths,
expected rounded metrics, toolchain pins, and upstream corpus commits.

Fast artifact verification:

    mise install && mise research:whatkey-headline-verify

Full headline rerun:

    mise install
    mise research:whatkey-headline-rerun -- --yes

The check reads these report files and compares their rounded metrics with the
published table:

    WhatKey (causal, abstaining)

    music21 Temperley-Kostka-Payne

    music21 Krumhansl-Schmuckler
      test-iso-m21-krumhanslschmuckler/report.json
